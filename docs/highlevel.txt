A tool to lent any credit card - 

Requirement -
when the user book flight, shooping , we see many credit card offers…. But having all the credit card for a enduser is not viable solution. 
So we will lent our pool of credit cards to the user to avail those offers. 
The system will provide the users a given limit to use from the selected credit card.


High level Flow - 

1. The user logs in the tool/site- using google OAuth
2. Once logged in, Show the option to lend a credit card limit with three inputs.
3. First input -> Purchase Type : with values -> (Flight Booking, Online shopping, E-Comm, Bills, Rent, Others )
4. second input -> Select the list of all the credit card which we offer 
5. Third input -> let the user to put a valid integer number(min is 10rs INR to 1 Lakh)
6. A pay Now Button
6. and Just above the button,  show a expend/ collapse  section for our service charge brokeage(0.3% or 100rs whichever is higher).
7. Then embedd the payments Page setup- With only UPI.
8. Once the payment is successfull, Show the credit card Full Number, expiry and cvv.
9. - here for the backend logic, as we will get the payment OTP because we are the owner, 
here verify the payment amount with the amount which the user have requested if it is exactly the same, only then show the OTP.
else 
    Show the user with two option -    
        1. Instant refund (with warning that the user request wrong credit limit.)
        2. Invest the amount with (1% monthly return.)


low level->

Design Type -> Light weight Website
backend -> Python, Django
frontend -> HTML ,CSS , Bootstrap
DB-> mysql


if anyother tech Debt is missed here , we will proceed that in next iteration.
